# codeless

To get started, run:

```
deno run --allow-net --allow-run --unstable main.ts
```

Then you should be able to make an HTTP request like so:

```
> curl localhost:8080/raw.githubusercontent.com/patrickdevivo/codeless/main/examples/hello_x.ts

hello, world!
hello, patrick!
hello, deno!
hello, reader!
```
